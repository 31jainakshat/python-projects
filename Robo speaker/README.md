# Robo Speaker

A tiny macOS voice automation tool that uses the built-in `say` command to speak user-provided text.

## 🚀 Features

- Prompts the user for text input
- Speaks the entered text aloud using macOS `say`
- Supports quitting by entering `q`

## 🧰 Requirements

- macOS (uses the built-in `say` command)
- Python 3

## ▶️ Usage

```bash
python main.py
```

Then type what you want the computer to say and press Enter.

To exit, type `q` and press Enter.

## 📝 Notes

- This project is a simple demo of running shell commands from Python.
- You can extend it by adding text-to-speech options, saving spoken phrases, or adding a GUI.
